The overlay checks these extensions in this order: mp3, wav, ogg, m4a.

Recommended approach:

- Use CC0 or clearly redistributable royalty-free audio only.
- Keep each file short, around 0.2s to 1.5s.
- Normalize peak level so one sound is not much louder than the others.

Suggested sources to review manually before adding files:

- Pixabay sound effects
- Mixkit sound effects
- Freesound with an explicit CC0 or redistribution-friendly license filter
